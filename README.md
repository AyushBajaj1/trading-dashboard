# Trading Dashboard

Responsive dashboard UI for trading analytics built with Vite and web technologies.  
This project provides a base structure to visualize trading data and build interactive components.

---

##  Features

- Clean, minimal starter layout
- Ready for integration with real trading data
- Modular frontend structure

---

##  Technologies

This project uses:
- Vite
- JavaScript 
- Python (FastAPI)

---

##  Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/AyushBajaj1/trading-dashboard.git
cd trading-dashboard
```

2. Install dependencies
```
npm install
```

4. Start development server
```
npm run dev
```

5. Open in browser

By default, Vite serves at:

http://localhost:5173
